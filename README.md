# photo-app-project

For the course Web Applications (CIS 142) at Stanford, I created a fully functional photo-app from scratch in 4 weeks. I built the frontend with React following the Model-View-Controller (MVC) decomposition, implemented capabilities to fetch data from a server with AJAX, and set up the database with MongoDB.

The web app supports all basic functionalities for users to log in and out, register, view their friendsâ€™ photos, like and comment on photos, and upload photos. It includes a user profile page and allows users to favorite photos with a user-specific list of favorites page. Users can also remove favorited photos from that list.
